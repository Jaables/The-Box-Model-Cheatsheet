The Box Model Cheatsheet

Tag

Name

Description
<width> Width Controls the width of an element. By default, it sets the width of the content area.
<height> Height Controls the height of an element. By default, it sets the height of the content area.
<padding> Padding The space between the contents of a box and the borders of a box is known as padding.
<border> Border Borders surround the content area and padding of an element. The color, style, and thickness of a border can be set with CSS properties.
<margin> Margin Margin refers to the space directly outside of the box. The margin property is used to specify the size of this space.
<overflow> Overflow The overflow property controls what happens to content that spills, or overflows, outside its box. The most commonly used values are:
  • <hidden>: when set to this value, any content that overflows will be hidden from view.
  • <scroll>: when set to this value, a scrollbar will be added to the elementâ€™s box so that the rest of the content can be viewed by scrolling.
  • <visible>: when set to this value, the overflow content will be displayed outside of the containing element. Note, this is the default value.
